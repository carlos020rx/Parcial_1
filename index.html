Paso a paso GitIgnore

Paso 1: Crear un archivo .gitignore

  1. Abre tu proyecto en tu editor de código o terminal.
  2. Crea un archivo llamado .gitignore en la raíz de tu repositorio (es decir, en la carpeta principal del proyecto).

Paso 2: Editar el archivo .gitignore

  1. Abre el archivo .gitignore en tu editor de texto.
  2. Añade las rutas de los archivos o carpetas que deseas excluir del seguimiento de Git.
    • Para ignorar carpetas agrega un "/" al final del nombre de la carpeta.
    • Para ignorar archivos de algun tipo usa "*" al final.
    • Para eliminar un archivo solo pon el nombre.

Paso 3: Guardar y confirmar los cambios

  1. Guarda el archivo .gitignore después de editarlo.
  2. Confirma que Git ha dejado de rastrear los archivos o carpetas especificados en .gitignore.
  3. Si ya has rastreado un archivo antes de agregarlo a .gitignore, Git seguirá rastreándolo. Para dejar de rastrearlo, debes eliminarlo del índice de Git.

Paso 4: Verificar que funciona

  1. Verifica que los archivos o carpetas mencionados en .gitignore no aparezcan al ejecutar.
